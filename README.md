# Portfolio

A personal portfolio site built with React, TypeScript, Vite, and Tailwind CSS.

The current experience includes:

- Hero section with interactive shader background (ColorBends + DotField)
- Projects section with project cards/list
- About section with profile content + work timeline
- Contact section scaffold

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- React Bits
  - Three.js (for shader-driven visuals)
  - Framer Motion / Motion / GSAP (animation tooling)
